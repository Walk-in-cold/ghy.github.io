五策略博弈动力学仿真实验

用到的数据：

分类20210718.xlsx：对其中的双复数数据进行处理，需要处理的数据后有标号，为K列。

manifold5try.mat：存放支付矩阵等数据，共491个五策略支付矩阵，通过main_of_get_eigen.m与对应标号来读取。

用到的函数

  1. 函数get_payoffmatrix_from_index

参数R0：所有的支付矩阵

Id：需要的支付矩阵序号

返回对应支付矩阵

% Program: Socexp, WZJ, 20210721

% Input: R --- the payoff array,

% id --- the number of the game

% Output: the 5x5 payoff matrix

function retM5x5 = get_payoffmatrix_from_index(Ro,id)

j=(id-1)*5;

retM5x5 = Ro(j+1:j+5,:);

end

  1. 函数get_eigensystem_from_5x5

参数：一个5x5的payoff_matrix

返回值：

特征向量，特征值，全非0的点数

% Program: Socexp, WZJ, 20210721

% Input: payoff_matrix --- the payoff

% Output: eigen_vector,

% d_eigen_value,

% rest_point_no

% of the 5x5 payoff matrix

function [eigen_vector, d_eigen_value, rest_point_no] = ...

get_eigensystem_from_5x5(payoff_matrix)

eigen_vector=[];

d_eigen_value=[];

rest_point_no=1;

syms x1 x2 x3 x4 x5 real

Payoff_vector_field_F = payoff_matrix *[x1 x2 x3 x4 x5]';

mean_U = [x1 x2 x3 x4 x5] * Payoff_vector_field_F;

V_Eq_0 = [x1 x2 x3 x4 x5]'.*(Payoff_vector_field_F - mean_U);

Jac = jacobian(V_Eq_0);

S=vpasolve(V_Eq_0);

A=eval([S.x1 S.x2 S.x3 S.x4 S.x5]);

s3=prod(A,2);

id = find(s3>0);

if size(id,1)==1

x1=A(id,1);x2=A(id,2);x3=A(id,3);x4=A(id,4);x5=A(id,5);

D_Eq_at_NE = eval(Jac);

[eigen_vector, eigen_value] = eig(D_Eq_at_NE);

d_eigen_value =diag(eigen_value);

end

if size(id,1) > 1

rest_point_no = size(id,1);

end

end
